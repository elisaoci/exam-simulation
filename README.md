# Exam Simulation

Многопроцессная симуляция сдачи экзамена.

## Механика

- Экзаменаторы работают в отдельных процессах
- Студенты в общей очереди
- Ответы на вопросы по закону золотого сечения
- Влияние настроения экзаменатора на результат
- Обед экзаменаторов после 30 секунд работы

## Технологии

- Python 3.10+
- Multiprocessing

## Запуск

```bash
git clone https://github.com/elisaoci/exam-simulation.git
cd exam-simulation
python main.py
```

## Выходные данные

- Интерактивная таблица во время экзамена
- Итоговая статистика по завершении:
· результаты студентов
· статистика экзаменаторов
· лучшие студенты и экзаменаторы
· студенты к отчислению
· лучшие вопросы
· вердикт (удался/не удался)
